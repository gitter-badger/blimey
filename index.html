Blimey (in development)

Fast, cross platform, 3D game engine for .NET and Mono.




Getting Started

You can grab the source code from github here https://github.com/blimey. The source is currently spread across a number of standalone git repositories, however this may change in the future. A good place to start would be to clone https://github.com/blimey/cor-demo which is a very basic demo showcasing Blimey's cross platform capablilties (remember to pull in the repo's submodules): 

$ git clone --recursive git@github.com:blimey/cor-demo.git

Once you are comfortable with the cor-demo you may want to move on to looking at https://github.com/blimey/blimey-demo which demonstrates more of the capabilties of Blimey. To run this demo you must build the demo's runtime data using the Blimey Asset Builder. 

$ git clone --recursive git@github.com:blimey/blimey-demo.git

Contributing

All contributions are welcome. Just fork and create a pull request on github.

To make it easier for us to pull in your changes, try to:

  • Make a new pull request for each logical change.
  • Make each pull request a single coherent commit. Use git rebase -i to squash commits as necessary.
